# 书籍信息节点 (Book Info Node)

## 功能说明

书籍信息节点负责获取书籍的详细信息，包括梗概、版本比较、导读、书评等。

## 主要职责

1. **获取详细信息** - 从数据源获取书籍详情
   - 书籍梗概/简介
   - 版本对比（不同出版社、译者等）
   - 导读推荐
   - 用户书评

2. **内容整理** - 组织和格式化信息

3. **内容生成** - 使用 LLM 生成自然语言响应

## 输入

从意图识别节点获取：
```python
slots = {
  "query": "版本比较",
  "book_title": "Python高效编程",
  "author": "张三",
  "pub_info": ["机械工业出版社", "中文版"]
}
```

## 查询类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 梗概 | 书籍简介和内容概述 | "告诉我《活着》讲了什么" |
| 版本比较 | 不同出版社或版本的对比 | "这本书有哪些版本？" |
| 导读 | 专业的书籍导读和分析 | "这本书怎么读？" |
| 书评 | 用户评价和专业书评 | "大家对这本书怎么看？" |

## 工作流程

```
获取查询类型和书籍信息
    |
    v
从数据源检索相关内容
    |
    v
使用 LLM 生成自然语言回复
    |
    v
返回格式化的响应
```

## 返回格式

```python
{
  "query_type": "梗概",
  "book_title": "Python高效编程",
  "content": "这本书讲述了如何编写高效的 Python 代码...",
  "source": "douban"
}
```
